# Age Calculator

## About

A Python project that calculates a user's exact age in years, months, and days using their date of birth and the current date.

### Why?

I decided to build this project because age calculations come up often in conversations with my friends. Through this project, I explored how dates are represented in Python and learned how to handle real-world cases such as leap years, invalid dates, and future dates.

## Features

1. Calculates age in years, months, and days.
2. Uses the current system date automatically.
3. Handles different month lengths (30/31 days) accurately.
4. Supports leap years.
5. Validates user input and rejects impossible dates.
6. Prevents users from entering birth dates in the future.
7. Allows users to re-enter their birthday until a valid date is provided.

## Technologies Used

* Python
* Datetime module
* Calendar module
* GitHub

## How to Run

1. Clone the repository.

2. Open the project folder in your terminal.

3. Run the program using:

   python3 age_calc.py

4. Enter your date of birth when prompted.
